Bravo, vous avez réussi à afficher cette page en utilisant la méthode GET !

La méthode GET est utilisée pour récupérer des données depuis le serveur. Dans ce cas, vous avez récupéré cette page HTML.

Comment fonctionne la méthode GET ?

Lorsque vous cliquez sur un lien ou que vous entrez une URL dans votre navigateur, une requête HTTP utilisant la méthode GET est envoyée au serveur.

Voici à quoi ressemble une requête HTTP GET simple :

GET /GET.html HTTP/1.1 Host: webserv.com:3434 User-Agent: Mozilla/5.0 (Windows NT 10.0; Win64; x64) Accept: text/html,application/xhtml+xml,application/xml Connection: keep-alive

Le serveur traite cette requête et renvoie une réponse contenant cette page HTML :

HTTP/1.1 200 OK Content-Type: text/html Content-Length: 5732 Date: Wed, 18 Apr 2023 14:28:53 GMT Server: Webserv 42 GET Method - Webserv 42 ...
