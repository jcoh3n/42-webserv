Redirection HTTP

Cette page démontre la redirection HTTP configurée dans notre serveur.

Vous allez être redirigé vers 42.fr dans 5 secondes...

Vers:

-- Logs du serveur --

La redirection est configurée dans notre fichier config.conf :

location /redirect {

    allowed_methods=GET

    return=301 https://www.42.fr

}
