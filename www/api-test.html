Créer un fichier

Utilisez cette méthode pour créer un nouveau fichier sur le serveur. Le serveur traite la requête POST via un script CGI et crée le fichier demandé.

DELETE

Supprimer un fichier

Utilisez cette méthode pour supprimer un fichier existant du serveur. Vous pouvez supprimer le fichier que vous venez de créer avec la méthode POST.
